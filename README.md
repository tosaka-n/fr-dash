# fr-dash
[日本語版](./README.js.md)  
punch in/out [freee](https://p.secure.freee.co.jp/) and post to Slack from cli  
## Initial setting
1. git clone && npm i
2. set environment
   - `$ cp .env.sample .env`
3. `npm link`
## Usage
  - punch in: `fr-dash in`
  - punch out: `fr-dash out`
  - begin break time: `fr-dash begin`
  - end break time: `fr-dash end`
